Executive Summary

Miles per gallon (MPG) is an important criteria for assessing and rating automobiles. It depends of various factors: fuel quality, weight, number of cylinders, engine efficiency, transmission efficiency, driver ability, road conditions, etc. This report tries to explain the difference between the effect of automatic and manual transmission on MPG.

Lets explore the data

  • Describe the dateset: The mtcars dataset of contains information on various cars set on 11 variables namely: mpg (Miles per gallon), cyl (Number of cylinders), disp (Displacement cubic inches), hp (Gross horsepower), drat Rear axle ration, wt (Weight 1000lbs), qsec (1/4 mile time), vs (V/S), am (Transmission; 0 = automatic; 1 = manual ), gear (Number of forward gears) and carb (Number of carburetors).

  • Identify the key variables: This report identifies the mpg and am variables as the key for the purpose of this report. Other variables particularly cyl, gear and drat would be considered for additional analysis.

  • Explore the dataset: The mean mpg for when grouped by automatic and manual transmission are 17.14737 and 24.39231 respectively. We can infer that on an average manual transmission is better than automatic transmission. See figure for illustration.
